Habitat-Lab is an open-source research platform from Meta AI for developing and evaluating embodied AI agents. Built on top of [Habitat-Sim](https://github.com/facebookresearch/habitat-sim), it provides a flexible Python framework for defining tasks, configuring sensors, training policies with RL and IL algorithms, and benchmarking agent performance at scale.

## Get started in 3 steps

<Steps>
  <Step title="Install Habitat-Lab">
    Create a conda environment and install the core library and optional baselines package.

    ```bash
    conda create -n habitat python=3.9 cmake=3.14.0
    conda activate habitat
    conda install habitat-sim withbullet -c conda-forge -c aihabitat
    pip install -e habitat-lab
    pip install -e habitat-baselines  # optional, for RL training
    ```
  </Step>
  <Step title="Run a task">
    Load a pre-defined task through the Gym interface and step through it with random actions.

    ```python
    import gym
    import habitat.gym

    env = gym.make("HabitatRenderPick-v0")
    observations = env.reset()

    terminal = False
    while not terminal:
        observations, reward, terminal, info = env.step(env.action_space.sample())
    ```
  </Step>
  <Step title="Train a policy">
    Launch PPO training using the habitat-baselines runner with a pre-built config.

    ```bash
    python -m habitat_baselines.run \
      --config-name=pointnav/ppo_pointnav_example.yaml
    ```
  </Step>
</Steps>

## Key features

- **Flexible task definitions** — Point navigation, object navigation, rearrangement, VLN, EQA, and custom tasks via the registry system
- **Diverse robot support** — Fetch, Spot, Franka, humanoids, and custom articulated agents with configurable sensors
- **Scalable training** — DD-PPO for distributed multi-GPU training, VER trainer for high-throughput rollouts
- **Hierarchical RL** — Two-layer HRL with PDDL-based planners and learned low-level skills
- **Gym-compatible** — Full OpenAI Gym interface so any RL library can interact with Habitat environments
- **Hydra configuration** — Composable YAML configs for tasks, agents, sensors, and training hyperparameters

<Note>
  Habitat-Lab requires Python 3.9+ and CMake 3.14+. GPU support is recommended for training but not required for environment interaction.
</Note>
